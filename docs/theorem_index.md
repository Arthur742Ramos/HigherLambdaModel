# Higher Lambda Model Theorem Index

This document is the repository's paper-to-Lean claim matrix for the current
`master` branch. Every major theorem or construction from the two source papers
appears below with a Lean location, a representative declaration name, and an
explicit status.

Unformalized claims are kept visible and marked `partial` or `missing` rather
than omitted.

Status legend:

- `done`: a direct Lean declaration matching the claim exists and the current
  repository builds.
- `partial`: the repository contains a weaker, chosen-data, low-dimensional, or
  otherwise incomplete shadow of the paper claim.
- `missing`: no corresponding declaration was found in the current repository.

## Remaining Paper-Level Gaps

1. Extend the semantic 3-cell interpretation from the current structural
   fragment to the full primitive syntactic 3-cell language. The structural
   fragment is now closed under reflexive left/right whiskering and
   interchange, and the direct primitive coverage now also reaches both
   directions of whisker symmetry together with the second interchange form,
   so the remaining work is the direct semantic treatment of pentagon.
2. Upgrade Example 4.2 from the current pointwise / omega-groupoid separation to
   the full interpreted β/η contraction argument at the chosen-point level. The
   repo now has a real continuous interpreter in
   `HigherLambdaModel/KInfinity/ContinuousSemantics.lean`, explicit witness
   terms and contraction proofs for the paper’s `(λz.xz) y` example in
   `HigherLambdaModel/KInfinity/Examples.lean`, arbitrary one-step βη soundness
   for the continuous semantics, equality-generated continuous witness objects
   for the specific β-side and η-side contractions, and a chosen-data package
   pairing those witnesses with `interp1Beta` and `interp1Eta`. The file now
   also makes the proof-relevance obstruction explicit: `Example42DirectWitness`
   preserves the β-vs-η direct-witness choice exactly because
   `betaEtaPaper_beta1_eq_eta1` shows the ordinary `NTerm1.redex` layer already
   collapses those two proofs. So the remaining work is to derive the chosen
   point assignment semantically in a proof-relevant way rather than merely
   postulating it as the current paper-facing packaging choice.
3. Replace the present low-dimensional ω-groupoid package with a direct
   all-dimensional constructive formalization, and use it to add genuinely new
   expansion results beyond the original papers.
